## 보안

Microsoft는 GitHub 조직의 모든 소스 코드 리포지토리를 포함하여 소프트웨어 제품 및 서비스의 보안을 중요하게 생각합니다.

**공개 GitHub 문제를 통해 보안 취약점을 보고하지 마십시오.**

보안 보고 정보, 위치, 연락처 정보 및 정책에 대해서는 [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)에서 Microsoft 리포지토리에 대한 최신 지침을 검토하세요.
